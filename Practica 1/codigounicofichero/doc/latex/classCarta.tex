\doxysubsection{Detailed Description}
Este archivo implementa los metodos y fuciones hayados en el archivo \doxylink{Carta_8hpp}{Carta.\+hpp}. 

Esta clase permite instanciar objetos de tipo carta donde se refleja el umero y el palo de la misma. En esta clase se permiten conultas y modificaciones a datos privados a demas de una nnnicializaciion distinta dependiendo de las necesidades del usuario.
